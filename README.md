# PySpark Cheatsheet

A standalone quick reference for PySpark with Databricks. Optimized for the **Databricks Certified Professional Data Engineer** exam.

No installation, just open the HTML file in a browser.

## Sections

| Tab | Contents |
|-----|----------|
| Read & Write | Delta, CSV, JSON, Parquet — read/write modes and options |
| Transforms | `select`, `filter`, `withColumn`, `explode`, UDFs, and more |
| Actions | `collect`, `count`, `show`, `toPandas` — operations that trigger execution |
| Aggregations | `groupBy`, `agg`, common aggregate functions |
| Joins | Join types, broadcast hints, avoiding shuffles |
| Window functions | `rank`, `lag`, `lead`, rolling aggregations |
| Schema & Types | `StructType`, `StructField`, casting, schema enforcement |
| Delta Ops | Merge (upsert), time travel, `OPTIMIZE`, `VACUUM`, Change Data Feed |
| Streaming | `readStream`, `writeStream`, triggers, checkpoints, watermarks |
| Perf Tips | Partitioning, caching, `EXPLAIN`, broadcast joins, Photon |
| RDD | `map`, `filter`, `reduceByKey`, persistence |
